编程的原则
---
1. 正交原则

   正交直角坐标系（几何），坐标轴互不影响
   
   HTML、CSS、JS 三者互不影响，即改变某一个不会影响其它两个。
   
   不要用 JS 去干涉 CSS
   
   `$('#div').show()` VS `$('#div').addClass('active')`
   
   内容、样式、行为分离的原则
   
2. 面向接口编程

   首要代码怎么被调用
   
   ```javascript
   button.onClick = function () {
     var dialog = new Dialog({
         title: 'Tips',
         content: 'Hi, this is a example.'
         buttons: {
             ok: {
                 text: '确定'，
                 callback: function () {
                     console.log('User clicked "OK".');
                 }
             },
             cancel: {
                 text: '取消',
                 callback: function () {
                     console.log('User clicked "CANCEL".');
                 }
             }
         }
     });
   }
   ```
